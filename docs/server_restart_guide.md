# 서버 재시작 가이드

## 🔄 서버 재시작 방법

### 현재 상황
- 서버 코드는 정상적으로 컴파일되었습니다 ✅
- 새로운 주소 검색 API 라우트가 추가되었습니다 ✅
- 하지만 서버가 재시작되지 않아서 새 라우트가 등록되지 않았습니다 ❌

### 해결 방법

#### Step 1: 현재 실행 중인 서버 중지
1. 서버가 실행 중인 터미널 창 찾기
2. `Ctrl + C`를 눌러 서버 중지

#### Step 2: 서버 재시작
```bash
cd C:\Users\thf56\Documents\medicalstandard\server
npm start
```

#### Step 3: 정상 시작 확인
다음 메시지가 보이면 성공:
```
🚀 API server running on port 8080 (development)
```

---

## ✅ 완료 후 테스트

서버가 재시작되면:
1. Flutter 앱에서 주소 검색 다시 시도
2. 정상 작동하는지 확인

---

## 🔍 문제가 계속되면

1. **서버 로그 확인**: 서버 터미널에서 에러 메시지 확인
2. **환경 변수 확인**: `server/.env` 파일에 API 키가 있는지 확인
3. **포트 확인**: 서버가 8080 포트에서 실행 중인지 확인
