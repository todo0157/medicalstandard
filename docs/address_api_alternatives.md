# 주소 검색 API 대안 가이드

## 📋 개요

네이버 클라우드 플랫폼 가입이 어려운 경우를 위한 대안 API 가이드입니다.

---

## 🎯 대안 1: 카카오 주소 검색 API (권장 대안)

### 장점
- ✅ **개인 회원가입 매우 쉬움** (네이버보다 간단)
- ✅ 사업자 등록증 불필요
- ✅ 개발자 문서가 잘 정리됨
- ✅ 무료 할당량: 일 10만 건 (월 약 300만 건)

### 가입 방법
1. **카카오 디벨로퍼스 접속**: https://developers.kakao.com
2. **카카오 계정으로 로그인** (네이버 계정보다 더 간단)
3. **내 애플리케이션** → **애플리케이션 추가하기**
4. **앱 이름 입력** 후 생성
5. **앱 키** → **REST API 키** 복사

### API 키 발급
- **REST API 키**만 있으면 바로 사용 가능
- 추가 인증이나 복잡한 절차 없음

### 코드 변경 필요
현재 네이버 지도 API로 구현되어 있지만, 카카오 API로 변경 가능합니다.

---

## 🎯 대안 2: 네이버 클라우드 플랫폼 개인 회원가입

### 개인 회원가입 가능
네이버 클라우드 플랫폼은 **개인 회원**으로도 가입할 수 있습니다.

### 가입 시 주의사항
1. 회원가입 화면에서 **"개인 회원"** 선택
2. **"기업 회원"**을 선택하면 사업자 등록증이 필요함
3. 개인 회원은 사업자 등록증 **불필요**

### 개인 회원가입이 안 될 때
- 네이버 클라우드 플랫폼 고객센터 문의
- 또는 카카오 API 사용 권장

---

## 🎯 대안 3: 공공데이터포털 주소 검색 API

### 장점
- ✅ **완전 무료** (할당량 제한 없음)
- ✅ 사업자 등록증 불필요
- ✅ 공공기관 API로 안정적

### 단점
- ⚠️ API 응답이 다소 느릴 수 있음
- ⚠️ 문서화가 상대적으로 부족

### 가입 방법
1. **공공데이터포털 접속**: https://www.data.go.kr
2. 회원가입 (일반 회원)
3. **"주소 검색 API"** 검색
4. 활용신청 후 승인 대기

## 💡 추천 순서

### 1순위: 카카오 주소 검색 API
- **이유**: 가입이 가장 쉽고, 개발자 친화적
- **적합한 경우**: 빠르게 시작하고 싶을 때

### 2순위: 네이버 클라우드 플랫폼 (개인 회원)
- **이유**: 안정성과 확장성
- **적합한 경우**: 개인 회원가입이 가능할 때

### 3순위: 공공데이터포털
- **이유**: 완전 무료
- **적합한 경우**: 할당량 걱정 없이 사용하고 싶을 때

## 🔄 카카오 API로 전환하는 방법

현재 코드를 카카오 API로 변경하려면:

1. **서버 코드 수정** (`server/src/routes/address.routes.ts`)
   - 네이버 API 호출 부분을 카카오 API로 변경

2. **환경 변수 변경** (`server/.env`)
   - `NAVER_MAP_CLIENT_ID` → `KAKAO_REST_API_KEY`

3. **API 엔드포인트 변경**
   - 네이버: `https://naveropenapi.apigw.ntruss.com/...`
   - 카카오: `https://dapi.kakao.com/v2/local/search/address.json`

## ❓ 자주 묻는 질문

### Q: 개인 회원가입이 정말 가능한가요?
**A**: 네, 가능합니다. 네이버 클라우드 플랫폼 회원가입 화면에서 "개인 회원"을 선택하면 사업자 등록증 없이 가입할 수 있습니다.

### Q: 개인 회원으로도 AI·NAVER API를 사용할 수 있나요?
**A**: 네, 가능합니다. AI·NAVER API는 개인 회원도 사용할 수 있습니다.

### Q: 카카오 API로 변경하는 게 더 쉬운가요?
**A**: 네, 카카오 디벨로퍼스 가입이 더 간단하고 빠릅니다. 카카오 계정만 있으면 바로 시작할 수 있습니다.

### Q: 두 API를 모두 사용할 수 있나요?
**A**: 네, 가능합니다. 하이브리드 접근 방식으로 네이버를 주로 사용하고, 카카오를 백업으로 사용할 수 있습니다.
