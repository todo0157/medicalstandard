# 구현 제한 사항 및 사용자 개입 필요 항목

## 🔴 제가 직접 작업할 수 없는 항목

### 1. **외부 서비스 API 키 발급 및 계정 생성**

#### 카카오 주소 검색 API (항목 4)
- ❌ **불가능**: 카카오 개발자 계정 생성 및 API 키 발급
- ✅ **가능**: API 키를 받으면 코드 구현 가능
- **사용자 작업 필요**:
  1. https://developers.kakao.com 접속
  2. 애플리케이션 등록
  3. REST API 키 발급
  4. 환경 변수에 키 추가

#### 이미지 스토리지 서비스 (항목 1)
- ❌ **불가능**: Cloudinary/AWS S3 계정 생성 및 API 키 발급
- ✅ **가능**: API 키를 받으면 코드 구현 가능
- **사용자 작업 필요**:
  - Cloudinary: https://cloudinary.com 회원가입 → API 키 발급
  - AWS S3: AWS 계정 생성 → IAM 키 발급
  - 환경 변수에 키 추가

#### Firebase Cloud Messaging (항목 2 - 푸시 알림)
- ❌ **불가능**: Firebase 프로젝트 생성 및 설정
- ✅ **가능**: 프로젝트 설정 파일을 받으면 코드 구현 가능
- **사용자 작업 필요**:
  1. Firebase Console에서 프로젝트 생성
  2. FCM 설정
  3. `google-services.json` (Android) / `GoogleService-Info.plist` (iOS) 다운로드
  4. 프로젝트에 파일 추가

#### 결제 게이트웨이 (항목 9)
- ❌ **불가능**: 토스페이먼츠/이니시스 등 계정 생성 및 API 키 발급
- ✅ **가능**: API 키를 받으면 코드 구현 가능
- **사용자 작업 필요**:
  - 사업자 등록증 필요 (실제 결제 서비스)
  - 테스트 모드 API 키 발급
  - 환경 변수에 키 추가

### 2. **실제 콘텐츠 작성**

#### 고객지원 & 법적 고지 (항목 12)
- ❌ **불가능**: 실제 FAQ 내용, 이용약관, 개인정보처리방침 작성
- ✅ **가능**:

  - 템플릿 제공
  - 콘텐츠를 입력받아 표시하는 기능
- **사용자 작업 필요**:
  - 법무팀 또는 전문가와 협의하여 실제 문서 작성
  - FAQ 내용 작성

  - UI 구조 및 화면 구현
  - 콘텐츠 관리 시스템 (CMS) 구조
  - 콘텐츠를 입력받아 표시하는 기능
- **사용자 작업 필요**:
  - 한의학 전문가와 협의하여 콘텐츠 작성
  - 또는 외부 콘텐츠 API 연동 결정

나머지 30%는:
- 외부 서비스 API 키 발급 (코드 구현은 가능, 키만 필요)
