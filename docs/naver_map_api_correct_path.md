# 네이버 지도 API 올바른 경로 안내

## ⚠️ 현재 화면 문제점

현재 보시는 화면에는 **지도 API (Geocoding)**가 보이지 않습니다.

화면에 보이는 서비스들:
- ❌ CLOVA Speech Recognition
- ❌ CLOVA Voice - Premium
- ❌ CAPTCHA (Image/Audio)
- ❌ Search Trend

**지도 API는 이 목록에 없습니다!**

---

## ✅ 올바른 경로

### 방법 1: AI·NAVER API에서 Maps 서비스 찾기

1. **현재 위치 확인**
   - 좌측 메뉴: "AI-NAVER API" → "Application" (현재 위치 ✅)

2. **서비스 목록에서 Maps 찾기**
   - 현재 화면의 "Service 선택" 섹션을 **아래로 스크롤**하세요
   - 또는 **"NAVER Services"** 섹션을 확장하세요
   - **"Maps"** 또는 **"Geocoding"** 서비스를 찾으세요

3. **만약 Maps가 보이지 않는다면**
   - 다른 메뉴 경로를 시도해야 합니다

---

### 방법 2: 다른 메뉴 경로 시도

#### 옵션 A: Services 메뉴에서 직접 찾기

1. 좌측 메뉴에서 **"Services"** 클릭
2. **"AI·NAVER API"** 섹션 확장
3. **"Maps"** 또는 **"Geocoding"** 찾기

#### 옵션 B: 검색 기능 사용

1. 콘솔 상단의 **검색창** 사용
2. **"Geocoding"** 또는 **"지도 API"** 검색
3. 검색 결과에서 해당 서비스 선택

---

### 방법 3: 직접 URL 접근

지도 API는 다음 URL에서 직접 접근할 수 있습니다:

```
https://console.ncloud.com/naver-service/application/create
```

하지만 서비스 목록에 **"Maps"** 또는 **"Geocoding"**이 있어야 합니다.

---

## 🔍 지도 API 서비스 이름 확인

네이버 클라우드 플랫폼에서 지도 API는 다음 이름으로 표시될 수 있습니다:

- ✅ **"Maps"** (가장 일반적)
- ✅ **"Geocoding"** 
- ✅ **"지도 API"**
- ✅ **"주소 검색 API"**

---

## 📝 현재 화면에서 해야 할 일

### Step 1: 서비스 목록 스크롤
현재 화면의 **"Service 선택"** 섹션을 **아래로 스크롤**하여 더 많은 서비스를 확인하세요.

### Step 2: NAVER Services 섹션 확인
**"NAVER Services"** 섹션을 확장하거나 아래로 스크롤하여 **"Maps"** 또는 **"Geocoding"**을 찾으세요.

### Step 3: 다른 메뉴 시도
만약 여전히 보이지 않는다면:
1. 좌측 메뉴에서 **"Services"** → **"AI·NAVER API"** 클릭
2. 또는 콘솔 홈으로 돌아가서 다시 시작

---

## 🎯 대안: 카카오 API 사용

만약 네이버 지도 API를 찾기 어렵다면, **카카오 주소 검색 API**를 사용하는 것을 권장합니다:

### 카카오 API 장점
- ✅ 가입이 더 간단
- ✅ 지도 API가 명확하게 표시됨
- ✅ 개발자 문서가 잘 정리됨

### 카카오 API 가입 방법
1. https://developers.kakao.com 접속
2. 카카오 계정으로 로그인
3. "내 애플리케이션" → "애플리케이션 추가하기"
4. **"로컬 API"** 활성화
5. **REST API 키** 복사

## ❓ 문제 해결

### Q: 현재 화면에 Maps가 보이지 않아요
**A**: 
1. 서비스 목록을 아래로 스크롤해보세요
2. "NAVER Services" 섹션을 확장해보세요
3. 다른 메뉴 경로를 시도해보세요
4. 또는 카카오 API 사용을 고려해보세요

### Q: Maps 서비스를 찾을 수 없어요
**A**: 
- 네이버 클라우드 플랫폼 고객센터에 문의하세요
- 또는 카카오 API로 전환하는 것을 권장합니다
