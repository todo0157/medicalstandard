# 네이버 지도 API 구독 가이드

## 📋 현재 상태

화면을 보니:
- ✅ Application "hanbang" 등록 완료
- ✅ Geocoding API 등록 완료
- ✅ Reverse Geocoding API 등록 완료
- ⚠️ 구독 상태 확인 필요

---

## 🔑 Step 1: Subscription 메뉴로 이동

### 1-1. 좌측 사이드바에서 "Subscription" 클릭
현재 화면의 좌측 사이드바에서:
1. **"Maps"** 섹션이 확장되어 있음
2. **"Subscription"** 메뉴 클릭
   - "Application" 아래에 있는 "Subscription" 메뉴

---

## 📝 Step 2: 구독 상태 확인

### 2-1. Subscription 화면 확인
"Subscription" 화면에서 다음을 확인:
- **구독 상태**: "구독 중" 또는 "구독 필요"
- **구독 플랜**: 무료 플랜 또는 유료 플랜
- **사용량**: 월 300만 건 무료 할당량

### 2-2. 구독이 안 되어 있다면
1. **"구독하기"** 또는 **"Subscribe"** 버튼 클릭
2. 구독 약관 동의
3. 구독 완료

---

## 🎯 Step 3: 구독 완료 확인

### 3-1. 구독 상태 확인
- Subscription 화면에서 **"구독 중"** 또는 **"Active"** 상태 확인

### 3-2. Application 화면에서 확인
- "Application" 화면으로 돌아가기
- API 사용량이 정상적으로 표시되는지 확인

---

## ⚠️ 주의사항

### 무료 플랜
- **월 300만 건**까지 무료
- Geocoding: 월 300만 건
- Reverse Geocoding: 월 300만 건

### 구독이 필요한 경우
- 일부 고급 기능은 유료 구독 필요
- 하지만 기본 Geocoding/Reverse Geocoding은 무료 플랜으로 사용 가능

---

## 🔍 문제 해결

### Q: "Subscription" 메뉴가 보이지 않아요
**A**: 
1. 좌측 사이드바에서 "Maps" 섹션 확장 확인
2. "Subscription"이 "Application" 아래에 있는지 확인
3. 페이지 새로고침

### Q: 구독 버튼이 없어요
**A**: 
- 이미 구독되어 있을 수 있습니다
- Subscription 화면에서 구독 상태 확인

### Q: 구독 후에도 401 에러가 발생해요
**A**: 
1. 서버 재시작 확인
2. API 키가 정확한지 확인
3. Application이 활성화되어 있는지 확인

---

## 📊 구독 완료 후

구독이 완료되면:
1. ✅ 서버 재시작
2. ✅ 주소 검색 테스트
3. ✅ 정상 작동 확인

---

## 💡 팁

- 사용량은 "Usage" 메뉴에서 확인할 수 있습니다
- 구독은 자동으로 갱신됩니다
